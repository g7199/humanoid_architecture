## Installation
### PyOpenGL
PyOpenGL은 다음과 같이 설치 할 수 있다.
$ pip install PyOpenGL PyOpenGL_accelerate

설치 과정에서 문제가 생긴다면 직접 코드를 받아서 설치하는 방법을 추천한다.

코드 다운로드
git clone https://github.com/mcfletch/pyopengl.git

설치
$ cd pyopengl
$ pip install -e .
$ cd accelerate
$ pip install -e .
